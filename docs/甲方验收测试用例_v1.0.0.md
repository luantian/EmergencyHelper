# 应急事件管理 App 验收测试用例（V1.0.0）

## 1. 文档信息

| 项目 | 内容 |
|---|---|
| 文档版本 | v1.0.0 |
| 适用版本 | Android 客户端 v1.0.0 |
| 测试类型 | 功能验收 / 业务流程验收 / 兼容性回归 |
| 目标系统 | Android 8.0 及以上 |
| 服务器地址 | `http://47.104.73.59:93` |

## 2. 测试范围

- 登录与鉴权
- 工作台（天气、统计、入口权限）
- 事件上报、事件列表、事件详情、动态、反馈、转派、办结
- 衍生风险上报、列表、详情、动态
- 通讯录（树结构、搜索、电话）
- 消息中心（列表、已读、角标）
- 重点点位（列表+地图联动）
- 附件（图片/视频上传、压缩、预览、删除）
- 推送（TIMPush 离线通知）
- 音视频通话（TUICallKit）

## 3. 测试环境建议

- 终端A：华为（HarmonyOS）  
- 终端B：小米/红米（Android）  
- 网络：Wi-Fi + 4G/5G 各一次  
- 账号：至少 2 个有效账号（用于转派、通话、推送验证）
